Glitch to Trello

This project is a first attempt at creating a system for automatically keeping track of Glitch projects in Trello using the Glitch API and the Trello API. All it currently can do is create a Trello board using the form below. It is written in CoffeeScript and you can view the source here. The boards for this proof of concept are created in the PatrickWeaver@Glitch Team on Trello.
